# Número Secreto

Juego de adivinanza de número secreto desarrollado con HTML, CSS y JavaScript puro.

## Descripción

El jugador debe adivinar un número secreto generado aleatoriamente entre **1 y 10**. El juego proporciona retroalimentación en cada intento hasta que el jugador lo descubra.

## Cómo jugar

1. Ingresa un número entre 1 y 10 en el campo de entrada.
2. Haz clic en **Intentar** para verificar tu respuesta.
3. El juego te indicará si el número secreto es mayor o menor.
4. Cuando adivines, haz clic en **Nuevo juego** para reiniciar.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts: [Chakra Petch](https://fonts.google.com/specimen/Chakra+Petch) y [Inter](https://fonts.google.com/specimen/Inter)
